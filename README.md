# doesman1227.github.io

個人サイトのハブ（トップページ）。GitHub Pages のユーザーサイトとして `https://doesman1227.github.io/` で公開しています。

各プロジェクトは同じドメインのサブパスで配信されます：

- `/mindshift/` — MindShift ／ マインドフルネスを学びなおすノート
- `/quantum-zukan/` — 量子力学ふしぎ図鑑

独自ドメインを設定した場合は、このリポジトリに `CNAME` を置くことで、全プロジェクトがそのドメインのサブパス（例：`https://example.com/mindshift/`）で配信されます。
