# doesman1227.github.io

個人サイトのハブ（トップページ）。GitHub Pages のユーザーサイトとして `https://doesman1227.github.io/` で公開しています。

各プロジェクトは同じドメインのサブパスで配信されます：

- `/mindfulness/` — マインドフルネス ／ まとめノート
- `/quantum/` — 量子力学 ／ まとめノート

独自ドメインを設定した場合は、このリポジトリに `CNAME` を置くことで、全プロジェクトがそのドメインのサブパス（例：`https://example.com/mindfulness/`）で配信されます。
